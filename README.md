# WotVehicleNameEditor

Utility for editing vehicles names found in .mo files in World of Tanks

![preview.png](https://github.com/Svensk2137/WotVehicleNameEditor/blob/main/preview.png)

# Preperation
1. Move `*_vehicles.mo` of the nation you want to edit from `res/text/lc_messages/`  to `res_mods/<current_version>/text/lc_messages/`

# Running
1. Choose folder `res_mods/<current_version>/text/lc_messages/`
2. Select nation file
3. Select vehicle
4. Change name
5. Save changes
6. Enjoy :D
